Menu bar reads 24ms when the link is healthy, and a when it times out. Both drawn in white — never a color.

One number. Two states.

No dashboards, no accounts, no noise. Just the round-trip time to the host you care about.

Live latency

Real round-trip time to your host, refreshed on an interval and shown right in the menu bar.

Clear timeout signal

When a packet doesn't come back, the number becomes a caution triangle. No guessing, no color.

Any host

Point it at any IP or hostname: your SSH box, a gateway, a game server. Remembered across launches.

Adjustable interval

Ping every 0.1s to 10s: tight when you're debugging, relaxed when you're not.

Launch at login

One toggle, using the native login-item service. It's just there every time you sit down.

Zero dependencies

A single Swift file, one universal binary, about 30 KB. No Python, no SwiftBar, no packages.

Running in under a minute.

It's unsigned, so macOS asks once. Here's the whole dance.

  1. Download & unzip

    Grab LatencyMon-universal.zip from the latest release and unzip it.

  2. Let macOS open it

    Move the app to /Applications, launch it once, then open System Settings → Privacy & Security and click Open Anyway. Or run xattr -dr com.apple.quarantine LatencyMon.app.

  3. Set your host

    Click the menu-bar item → Set IP…, enter your SSH host, and you're watching.
